# Obfuscator

Dùng để làm rối code Python đơn giản.

## Yêu cầu

- Python 3.12

## Cài đặt & chạy

```bash
git clone https://github.com/raphaelkissu/obfuscator.git
cd obfuscator
python main.py
```

## Cách dùng

```bash
python main.py
```

Sau đó nhập:

```
File đầu vào  : <example.py>
File đầu ra   : <example_obf.py>
```

Mỗi lần chạy sẽ cho output khác nhau.
